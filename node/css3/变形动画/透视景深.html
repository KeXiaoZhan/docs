# 透视景深

# perspective

  • 使用 perspective 来控制元素的透视景深
  • perspective 规则为舞台元素控制景深， perspective 属性为控制单个元素

# 舞台透视

perspective 规则（注意是规则，就是单独使用perspective）用于将父级整个做为透视元素，会造成里面的每个子元素的透视是不一样的。就像现实中摆一排杯子，是使用统一透视的，每个杯子的透视不一样，造成有大有小。(注意，使用规则父级本身是不受影响的，只会影响孩子,搭配3D透视可以看出孩子的层次感)

<style>
    article {
        margin: 0 auto;
        margin-top: 150px;
        width: 400px;
        height: 200px;
        position: relative;
        border: solid 5px silver;
        perspective: 200px;
    }

    article div {
        width: 100px;
        height: 100px;
        background: blueviolet;
        box-sizing: border-box;
        margin-right: 80px;
        float: left;
        transform: rotateY(60deg);
    }
</style>

<article>
	<div></div>
	<div></div>
</article>

# 单独透视

perspective 函数用于为元素设置单独透视，下面是为元素单独设置透视参数，每个元素的透视效果是一样的。（使用函数会影响本身，也会影响孩子）

article div {
    width: 100px;
    height: 100px;
    background: blueviolet;
    box-sizing: border-box;
    margin-right: 80px;
    float: left;
    transform: perspective(100px) rotateY(60deg);
}
